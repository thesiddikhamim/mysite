# AGENTS.md

## Project Overview

Static portfolio website deployed to GitHub Pages: https://thesiddikhamim.github.io/mysite

## Tech Stack

- Plain HTML/CSS/JavaScript
- No build tools, no package manager
- Font Awesome for icons (CDN)
- Google Fonts for typography
- Google Analytics (gtag.js)

## Directory Structure

- `index.html` - Main entry point
- `css/` - Stylesheets (style.css, hero.css, navbar.css)
- `script.js` - Main JavaScript
- `projects/` - Sub-project pages (income-inequality.html, gini-coefficient.html)
- `certificate/` - Certificate images
- `photos/` - Profile and other images
- `font/` - Custom font files
- `books/` - Book-related pages
- `animaitons/` - Animation assets

## Development

No build commands needed. Edit files directly and open `index.html` in a browser.

## Deployment

Push to `main` branch - GitHub Pages auto-deploys from root.
